# Codex Skills

Reusable skills for Codex workflows.

## Codex Skills

- `daily-ai-brief`: Produces a ranked Daily AI and Agentic Engineering HTML brief with live web research, source-quality rules, recent-run dedupe, and a portable run log.
- `orchestrate-projects`: Coordinates long-running, multi-milestone projects through conversational discovery, evidence-gated roadmaps, delegated execution, and milestone audits.

Each Codex skill lives in its own folder with a `SKILL.md` entrypoint and any supporting references.
